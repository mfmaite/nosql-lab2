4. Instrucciones para Restaurar el Volumen de Jenkins:

Crear el volumen jenkins_home:
docker volume create nosql-lab2_jenkins_home

Restaurar el contenido del volumen:
docker run --rm -v nosql-lab2_jenkins_home:/data -v $(pwd):/backup busybox tar xvf /backup/jenkins_home_backup.tar -C /

// Este le funciono a Mei
docker run --rm -v nosql-lab2_jenkins_home:/data -v $(pwd)/backup:/backup busybox tar xvf /backup/jenkins_home_backup.tar -C /
